# **App Name**: NoorRecite

## Core Features:

- Quran Text Display: Display the full Quran text with multiple selectable translations.
- Audio Recitation: Enable audio recitation of the Quran with options for different reciters.
- Personal Reading Statistics: Track the user's reading progress, including verses read, time spent reading, and consistency.
- Unlockable Achievements: Implement a system of unlockable achievements based on reading milestones, to encourage continued engagement.
- Advanced Search: Provide an advanced search functionality to quickly find specific verses or topics within the Quran.

## Style Guidelines:

- Primary color: Soft, calming green (#A7D9A5) to evoke tranquility and spiritual connection.
- Background color: Off-white (#F5F5F5) for a clean and peaceful reading experience.
- Accent color: Warm gold (#D4AF37), inspired by Islamic art and architecture, to highlight important elements and actions.
- Body font: 'Alegreya', a serif, for readability in longer passages.
- Headline font: 'PT Sans', a sans-serif, for headings and titles, providing a modern contrast to the body text.
- Use Islamic mosaic patterns and minimalist icons representing chapters and verses for easy navigation.
- Ensure a clean, spacious layout that promotes focus and minimizes distractions, using clear visual hierarchy.
